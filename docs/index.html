    Logger for browser and node

    nightingale

    To see the full documentation, go to nightingalejs/nightingale

    To see the Logger API, go to [docklets][docklets-url] or http://nightingalejs.github.io/nightingale-logger

    Install

    npm install --save nightingale-logger

    Levels

    nightingale-levels

    Usage

    import Logger from 'nightingale-logger';

const logger = new Logger('mylib');

logger.info('This is a log');
logger.warn('This is a warning !');
logger.success('It works !');
